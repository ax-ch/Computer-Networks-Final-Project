# Computer-Networks-Final-Project

A Python implementation of a Client–Proxy–Server architecture developed for the Computer Networks final project at Telkom University.

This repository contains the complete project deliverables, including the source code for the Client, Proxy Server, and Web Server, website resources, custom HTTP error pages, Wireshark packet captures, troubleshooting records, demonstration materials, and supporting documentation used throughout development and testing.

The project demonstrates low-level socket programming concepts such as TCP-based HTTP communication, proxy caching, UDP-based QoS evaluation, and multithreaded request handling. Performance and network behaviour were further validated through QoS measurements and Wireshark traffic analysis under different load conditions.

Features

* Client–Proxy–Server architecture
* HTTP communication using TCP sockets
* Proxy forwarding and caching
* Cache HIT and Cache MISS mechanisms
* UDP Echo service for QoS evaluation
* Multithreaded concurrent request handling
* Custom HTTP error pages
* Wireshark-based packet analysis

Repository Contents

* Source code for the Client, Proxy Server, and Web Server
* Website pages and static assets
* Custom status/error pages
* Wireshark capture files
* Demo scripts and notes
* Troubleshooting and progress logs
* Supporting materials for the final report and presentation
